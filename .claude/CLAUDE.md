# Проект: Конкур (Funny-horse)

Игра в стиле Марио с лошадьми и наездниками. Браузерная, на vanilla JS + HTML5 Canvas.

## Файлы проекта
- `index.html` — точка входа
- `styles.css` — стили и UI
- `game.js` — вся игровая логика

## Субагенты (Skills)

Используй нужный skill в зависимости от задачи:

| Команда | Описание |
|---|---|
| `/backlog` | Управление задачами и приоритетами |
| `/frontend` | Игровая логика, Canvas, анимации |
| `/backend` | localStorage, API, рекорды |
| `/ui-design` | Визуальный стиль, цвета, макеты экранов |

## Стек
- Язык: JavaScript (ES6+), без фреймворков
- Рендер: HTML5 Canvas 2D
- Хранилище: localStorage (MVP)
- Деплой: GitHub Pages

## Правила
- Всегда отвечать на русском языке
- Код — без лишних комментариев
- Приоритет — рабочий минимум, потом полировка
